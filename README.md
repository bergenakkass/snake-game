# Snake Game (Python + Pygame)

##  Açıklama
Bu proje Python ve Pygame kullanılarak geliştirilmiş bir Snake oyunudur.

##  Özellikler
- Klavye ile kontrol (ok tuşları)
- Yem yedikçe büyüme
- Skor sistemi
- Game Over ekranı
- Restart özelliği
- High score kaydı
- Dinamik hız artışı

## Çalıştırma

```bash
pip install pygame
python main.py
